HLOOKUP DAN VLOOKUP

VLOOKUP

Fungsi pencarian pada data yang berbentuk KOLOM / VERTICAL

Rumus:
=VLOOKUP(lookup value, table array, col-index-num, range lookup)

1. LOOKUP VALUE adalah data yang akan dicocokkan.
2. TABEL ARRAY adalah tempat pencarian data dimulai dari colom yang berisi LOOKUP VALUE sampai kolom yang berisi hasil yang di cari COL INDEX NUM.
3. COL INDEX NUM adalah data pada kolom keberapa yang hendak diambil (hasil pencarian).
4. RANGE LOOKUP adalah nilai logika yang dimasukkan, jika diisi TRUE maka akan dicari sampai data terdekat (sebagian benar), sedang jika diisi FALSE maka akan dicari data yang persis sama.

HLOOKUP

HLOOKUP mencari value secara horizontal (dari kiri ke kanan), lalu menampilkan nilai lain yang sekolom dengan nilai tersebut sesuai yang diinginkan.

Rumus :
=HLOOKUP(lookup_value, table_array, row_index_num, range_lookup)

1.LOOKUP VALUE adalah data yang akan dicocokkan.
2. TABEL ARRAY adalah tempat pencarian data yang di inginkan
3. ROW INDEX NUM yaitu nomor urut row atau baris yang akan kita ambil nilainya sesuai dengan nomor kolom yang ada di tabel. Jika kita isi dengan nilai kurang dari 1 akan menghasilkan ERROR #VALUE!. Sedangkan jika kita isi dengan angka yang lebih besar dari jumlah kolom yang ada dalam tabel, akan menghasilkan error #REFF!.
4. RANGE LOOKUP adalah nilai logika yang dimasukkan, jika diisi TRUE maka akan dicari sampai data terdekat (sebagian benar), sedang jika diisi FALSE maka akan dicari data yang persis sama.

Pada ilustrasi di atas, kita akan mencari berapa jumlah penjualan mie instan varian Soto Ayam pada bulan Mei. Output dari rumus HLOOKUP pada cell D14 akan menghasilkan nilai 2200
